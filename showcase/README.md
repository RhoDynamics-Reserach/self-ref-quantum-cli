## 🎨 How to Showcase Your Agent

Sharing your matured agents allows other researchers to use your specialized "Cognitive Foundations" as a starting point for their own experiments.

### 1. Maturity Check
We recommend only submitting agents with a **Zeta ($\zeta$) > 2.5**. This ensures the manifold is stable enough for community use.

### 2. Exporting the DNA
In the `rhodynamics-cli`, run:
```bash
export <YourAgentName>
```
This will generate a `.rho.json` file.

### 3. Submission
1.  **Fork** the repo.
2.  Place your `.rho.json` file in the `showcase/assets/` directory.
3.  Add a row to the table above in `showcase/README.md`.
4.  Submit a **Pull Request**.

### ⚠️ A Note on Hardware
If your agent was evolved using a physical IBM QPU (e.g., `ibm_fez`), please mention the **Backend Name** in your PR description for added scientific prestige!
